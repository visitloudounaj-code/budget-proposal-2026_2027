## Structure

The proposal is organized as a single HTML landing page with tab-based navigation:

- **Executive Summary** — total investment, priority ranking, skill coverage and annual calendar
- **One tab per new conference** — cost breakdown, why Visit Loudoun should invest, how it relates to the role and what I'll bring back

Each conference tab links directly to the official event site for further detail.

## Strategic Context

Every conference in this proposal is selected to advance the **Elevate Loudoun 2026–2028** strategic plan — specifically the shift from day-tripper to overnight visitor focus, the response to AI-driven changes in destination search and the measurable economic-impact reporting stakeholders increasingly expect.
